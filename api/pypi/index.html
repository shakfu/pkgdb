    PyPI API¶

    Functions for interacting with the PyPI stats API and PyPI JSON API.

    Download Statistics¶

    fetch_package_stats ¶

    fetch_package_stats(
    package_name: str,
) -> PackageStats | None

    Fetch download statistics for a package from PyPI.

    Returns None if the package doesn't exist or the API is unreachable.
